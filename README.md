# CoinRailgun

一个基于nodejs的静态博客生成器


## Quick Start

**Install CoinRailgun**

```bash
npm install -g coinrailgun
```

**Initialize your blog**

```bash
crn init blog
cd blog
```

**Create new post**

```bash
crn new "Hello CRN"
```

**Build static files**

```bash
crn build
```

**Start local server**

```bash
crn server
```
